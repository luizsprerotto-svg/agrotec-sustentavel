# 🌾 Projeto Agrinho 2026

Este repositório contém o site desenvolvido para o **Programa Agrinho 2026**. O objetivo do projeto é explorar a relação entre sustentabilidade, cidadania e o uso de novas tecnologias no meio rural, promovendo a conscientização ambiental e o apoio à agricultura familiar.

O site foi estruturado de forma moderna, responsiva e com interações em JavaScript, ideal para apresentação e publicação na web através do GitHub Pages.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias web fundamentais:

* **HTML5:** Estruturação semântica de todo o conteúdo.
* **CSS3:** Design moderno, paleta de cores voltada à natureza (tons de verde) e total responsividade para dispositivos móveis.
* **JavaScript:** Interatividade com o usuário (rolagem suave e validação/simulação do formulário de contato).
